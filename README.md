Place The Rabbit Game 🐰


الوصف:
لعبة نصية ممتعة ومثيرة بلغة البايثون (Python)، هدفك فيها هو العثور على الأرنب المختبئ داخل شبكة عشوائية باستخدام ذكائك وتوقعاتك.


الميزات:
واجهة لعب بسيطة وتفاعلية عبر الطرفية (Terminal).
توليد عشوائي لموقع الأرنب في كل مرة تبدأ فيها اللعب.
تقييم لخطواتك لمعرفة مدى كفاءتك في الوصول للأرنب.
